# Smart_NetID

Award-winning C# application for digital identification and automated time recording.

## Overview
Smart NetID is a C# desktop application designed to replace physical ID cards and automate the recording of attendance and time. The system allows users to check in and out via a unique NetID, maintains a persistent attendance log, and provides administrators with a searchable student roster — all through a clean Windows Forms interface.

## Background
Smart NetID was developed as the final submission for a competitive C# workshop challenge. Participation was limited exclusively to workshop attendees, with approximately 30 competitors submitting entries. The project was awarded first place and was commended by the instructors for being an outstanding and fully functional submission.

## Duration
C# workshop took place every Saturday afternoon for 5 hours. 

## Technical Notes and Challenge
The original project was built as a console application in C#. The expanded version retains the same core logic whilst introducing a Windows Forms GUI and persistent data storage.
One notable challenge during development was the author's stronger background in Java. Certain design decisions during the workshop were initially approached with Java idioms in mind, which provided a useful opportunity to understand the distinctions between the two languages — particularly around event-driven GUI programming and .NET conventions.
